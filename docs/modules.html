pair

  • pair<Hook>(hook: Hook): FunctionComponent<PairedComponentProperties<Hook>>
    • Defined in pair.ts:42

    Creates a component with a function children that has the given hook in context.

    example
    const useCount = initialCount => {
        const [count, setCount] = useState(initialCount);

        return { onClick: () => setCount(count + 1), children: count };
    };

    const PairedCount = pair(useCount);

    const Component = ({ array = [] }) => (
        <ul>
            {array.map(key => (
                <PairedCount key={key}>
                    {usePairedCount => {
                        const props = usePairedCount(key);

                        return (
                            <li>
                                <button
                                    type="button"
                                    {...props}
                                />
                            </li>
                        );
                    }}
                </PairedCount>
            ))}
        </ul>
    );

    Type parameters

    • Hook: GenericFunction<ReadOnlyArray<never>, unknown>

    Parameters

    • hook: Hook

      Hook to be paired.

    Returns FunctionComponent<PairedComponentProperties<Hook>>

    Component that expects a function as children with the paired hook.
